# **VUE-WEBPACK-PROJECT**

## Usage
- 构建 : npm run build
- 开发环境： npm run startdev

## 工程介绍
- 实时刷新&热更新
    - 使用 `--watch`监听文件重新打包 和 `livereload`监听文件变动刷新浏览器
    - 使用`devServer`开启服务，异步接口代理到node服务，html路径重定向到devServer管理的路径（本工程采用此方法）
